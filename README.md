# resq
ResqApp Repository

# Installation

## Backend

Backend is an ZF2/Apigility based API REST service which provides all required data

To fully initialize backent type

    composer install

in the /backend directory


## Frontend
### Cordova installation

```
$ sudo npm install -g cordova
$ cd Resq
```

## Running

In Chrome browser:

```
$ cordova run browser
```
